Update docs/progress.json (or docs/progress.md) with current session state:
- current_task: what is being worked on
- status: in_progress / blocked / completed
- completed_steps: what was accomplished this session
- next_steps: what should happen next
- decisions_made: any decisions with rationale
- blockers: anything blocking progress
- last_updated: current timestamp

Then commit all changes with message: "save: [brief summary of session work]"
